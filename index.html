POMOWORK

COMBATIENDO EL
HIPER-ENFOQUE

Una solución tangible para la fatiga visual digital. Pomowork utiliza computación física y códigos lumínicos para restaurar tus ciclos circadianos de atención.

VER LA SOLUCIÓN

La Problemática

El "Síndrome Visual Informático" afecta al 70% de estudiantes. Usar el smartphone para descansar es contraproducente por los bucles de dopamina.

Nuestra Propuesta

Un dispositivo "Offline-First". Al exteriorizar el control del tiempo a un objeto físico (LEDs RGB), reducimos la carga cognitiva.

ALINEACIÓN CON ODS (ONU)

ODS 3: Salud y Bienestar

Reducción de la fatiga visual y el estrés digital mediante la promoción de pausas activas y desconexión programada.

ODS 4: Educación de Calidad

Mejora de los hábitos de estudio y la concentración profunda ("Deep Work") en entornos académicos.

ODS 9: Innovación e Infraestructura

Fomento del desarrollo de hardware libre (Open Source) y tecnologías de fabricación digital (Impresión 3D).

CÓDIGO FUENTE (C++)
/* POMOWORK - ARDUINO UNO */ #include <Wire.h> #include <LiquidCrystal_I2C.h> const int BTN_POWER = 2; const int BTN_MODE_25 = 3; const int BTN_MODE_20 = 4; const int PIN_RED = 9; const int PIN_GREEN = 10; const int PIN_BLUE = 11; const int PIN_BUZZER = 8; LiquidCrystal_I2C lcd(0x27, 16, 2); bool sistemaEncendido = false; int estadoActual = 0; unsigned long tiempoInicio = 0; unsigned long duracion = 0; int ultimoTiempoTrabajo = 25; const unsigned long TIEMPO_DESCANSO_TEST = 5000UL; unsigned long tiempoPrimerClickModo = 0; bool esperandoConfirmacionModo = false; void setup() { pinMode(BTN_POWER, INPUT_PULLUP); pinMode(BTN_MODE_25, INPUT_PULLUP); pinMode(BTN_MODE_20, INPUT_PULLUP); pinMode(PIN_RED, OUTPUT); pinMode(PIN_GREEN, OUTPUT); pinMode(PIN_BLUE, OUTPUT); pinMode(PIN_BUZZER, OUTPUT); lcd.init(); lcd.backlight(); apagarTodo(); } void loop() { if (digitalRead(BTN_POWER) == LOW) { delay(50); if (!sistemaEncendido) { encenderSistema(); while(digitalRead(BTN_POWER) == LOW); } else { bool dobleClick = false; unsigned long tiempoEsperaClick = millis(); while(digitalRead(BTN_POWER) == LOW); while (millis() - tiempoEsperaClick < 400) { if (digitalRead(BTN_POWER) == LOW) { dobleClick = true; while(digitalRead(BTN_POWER) == LOW); break; } } if (dobleClick) { apagarTodo(); } else { resetearAMenu(); } } } if (!sistemaEncendido) return; switch (estadoActual) { case 0: ponerColor(0, 255, 0); lcd.setCursor(0, 0); lcd.print("LISTO (TEST SEGS)"); lcd.setCursor(0, 1); lcd.print("BTN2:25s BTN3:20s"); if (digitalRead(BTN_MODE_25) == LOW) { iniciarPomodoro(25); } else if (digitalRead(BTN_MODE_20) == LOW) { iniciarPomodoro(20); } break; case 1: ponerColor(255, 165, 0); gestionarTemporizador("TRABAJANDO...", 2); verificarBotonesCambioModo(); break; case 2: ponerColor(0, 0, 255); gestionarTemporizador("DESCANSANDO...", 1); verificarBotonesCambioModo(); break; } } void iniciarPomodoro(int tiempo) { ultimoTiempoTrabajo = tiempo; duracion = tiempo * 1000UL; tiempoInicio = millis(); estadoActual = 1; esperandoConfirmacionModo = false; lcd.clear(); lcd.print("INICIANDO..."); tone(PIN_BUZZER, 1000, 200); delay(500); lcd.clear(); } void gestionarTemporizador(String titulo, int siguienteEstado) { if (esperandoConfirmacionModo) return; unsigned long tiempoTranscurrido = millis() - tiempoInicio; unsigned long tiempoRestante = 0; if (duracion > tiempoTranscurrido) { tiempoRestante = duracion - tiempoTranscurrido; } lcd.setCursor(0, 0); lcd.print(titulo); mostrarTiempo(tiempoRestante); if (tiempoRestante == 0) { alarmaFin(); if (siguienteEstado == 2) { duracion = TIEMPO_DESCANSO_TEST; tiempoInicio = millis(); estadoActual = 2; lcd.clear(); } else if (siguienteEstado == 1) { iniciarPomodoro(ultimoTiempoTrabajo); } } } void verificarBotonesCambioModo() { int botonPresionado = 0; if (digitalRead(BTN_MODE_25) == LOW) botonPresionado = 25; if (digitalRead(BTN_MODE_20) == LOW) botonPresionado = 20; if (botonPresionado > 0) { if (esperandoConfirmacionModo && (millis() - tiempoPrimerClickModo < 2000)) { iniciarPomodoro(botonPresionado); } else { esperandoConfirmacionModo = true; tiempoPrimerClickModo = millis(); lcd.clear(); lcd.setCursor(0, 0); lcd.print(" CAMBIAR MODO? "); lcd.setCursor(0, 1); lcd.print(" PULSA OTRA VEZ "); tone(PIN_BUZZER, 500, 100); delay(500); } } if (esperandoConfirmacionModo && (millis() - tiempoPrimerClickModo >= 2000)) { esperandoConfirmacionModo = false; lcd.clear(); } } void mostrarTiempo(unsigned long ms) { unsigned long segundosTotales = ms / 1000; int minutos = segundosTotales / 60; int segundos = segundosTotales % 60; lcd.setCursor(5, 1); if (minutos < 10) lcd.print("0"); lcd.print(minutos); lcd.print(":"); if (segundos < 10) lcd.print("0"); lcd.print(segundos); } void ponerColor(int rojo, int verde, int azul) { analogWrite(PIN_RED, rojo); analogWrite(PIN_GREEN, verde); analogWrite(PIN_BLUE, azul); } void encenderSistema() { sistemaEncendido = true; lcd.backlight(); ponerColor(0, 255, 0); lcd.clear(); lcd.print("HOLA!"); delay(500); resetearAMenu(); } void resetearAMenu() { estadoActual = 0; esperandoConfirmacionModo = false; lcd.clear(); lcd.print("RESETEANDO..."); ponerColor(0, 255, 0); delay(800); lcd.clear(); } void apagarTodo() { lcd.clear(); lcd.print("APAGANDO..."); delay(500); lcd.noBacklight(); ponerColor(0, 0, 0); sistemaEncendido = false; lcd.clear(); } void alarmaFin() { for(int i=0; i<3; i++){ tone(PIN_BUZZER, 2000); delay(200); noTone(PIN_BUZZER); delay(100); } }

EVIDENCIA DEL PROCESO

PASO 01

Ideación y Bocetos

Definición de la interfaz física mediante bocetos. Decisión de usar feedback lumínico RGB.

PASO 02

Pruebas en Protoboard

Montaje inicial del circuito para validar el LCD I2C y los botones antes de soldar.

PASO 03

Programación del Firmware

Desarrollo de la máquina de estados en C++ y lógica de botones avanzada.

PASO 04

Modelado 3D (CAD)

Diseño de la carcasa en software CAD, asegurando tolerancias para los componentes.

PASO 05

Impresión y Postprocesado

Fabricación de la carcasa en PLA y lijado para un acabado final profesional.

PASO 06

Ensamblaje Final y Test

Integración de la electrónica en la carcasa y pruebas de usuario finales.
